# FesochiRelish — Portfolio Catering Site (Mock Ordering)

**Project purpose:** A professional-looking portfolio website for a catering brand.  
This repo shows frontend UX, an interactive gallery, and a simulated ordering flow (mock) to demonstrate practical DevOps/Web engineering skills.

## What’s included
- `index.html`, `style.css`, `script.js` — clean, modern frontend
- `images/` — place your images with exact filenames:
  - `relish.png` (logo)
  - `hero.jpeg` (hero background)
  - `eg.jpeg, egu.jpeg, egus.jpeg, egusi.jpeg, so.jpeg, sou.jpeg, soup.jpeg, chi.jpeg, chid.jpeg, chidi.jpeg, chidim.jpeg, chidima.jpeg`
- `.gitignore` — keep repo clean
- Order log stored locally; "Download Orders Log" button exports JSON of mock orders.
